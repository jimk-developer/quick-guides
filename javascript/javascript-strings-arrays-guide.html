üìù Working with Strings

Strings are immutable in JavaScript - they cannot be changed, only replaced. String methods return new strings rather than modifying the original.

String Properties and Basic Info

let text = "Hello World"; // Get string length console.log(text.length); // 11 // Access characters by index console.log(text[0]); // "H" console.log(text.charAt(0)); // "H" // Get character code (Unicode value) console.log(text.charCodeAt(0)); // 72

Searching in Strings

Method Description Returns
indexOf(str) Find first occurrence Index or -1
lastIndexOf(str) Find last occurrence Index or -1
includes(str) Check if contains true/false
startsWith(str) Check if starts with true/false
endsWith(str) Check if ends with true/false
let text = "The quick brown fox jumps over the lazy dog"; // Find substring console.log(text.indexOf("fox")); // 16 console.log(text.lastIndexOf("the")); // 31 // Check if contains console.log(text.includes("quick")); // true console.log(text.startsWith("The")); // true console.log(text.endsWith("dog")); // true // Find with starting position console.log(text.indexOf("the", 10)); // 31 (starts search at index 10)

Modifying Strings

let text = " Hello World "; // Change case console.log(text.toUpperCase()); // " HELLO WORLD " console.log(text.toLowerCase()); // " hello world " // Remove whitespace console.log(text.trim()); // "Hello World" console.log(text.trimStart()); // "Hello World " console.log(text.trimEnd()); // " Hello World" // Replace text let str = "I like cats. Cats are great."; console.log(str.replace("cats", "dogs")); // Replaces first only console.log(str.replaceAll("cats", "dogs")); // Replaces all // Repeat string console.log("Ha".repeat(3)); // "HaHaHa" // Pad strings console.log("5".padStart(3, "0")); // "005" console.log("5".padEnd(3, "0")); // "500"

Extracting Substrings

let text = "JavaScript"; // slice(start, end) - extracts from start to end console.log(text.slice(0, 4)); // "Java" console.log(text.slice(4)); // "Script" console.log(text.slice(-6)); // "Script" (negative = from end) // substring(start, end) - similar to slice, no negative console.log(text.substring(0, 4)); // "Java" console.log(text.substring(4)); // "Script" // substr(start, length) - DEPRECATED, use slice instead console.log(text.substr(0, 4)); // "Java" console.log(text.substr(4, 6)); // "Script"
üí° Best Practice: Use slice() for extracting substrings. It's more flexible and supports negative indices.

Splitting and Joining

// Split string into array let sentence = "The quick brown fox"; let words = sentence.split(" "); console.log(words); // ["The", "quick", "brown", "fox"] // Split into characters let chars = "Hello".split(""); console.log(chars); // ["H", "e", "l", "l", "o"] // Split with limit let limited = sentence.split(" ", 2); console.log(limited); // ["The", "quick"] // Join array into string let arr = ["JavaScript", "is", "awesome"]; console.log(arr.join(" ")); // "JavaScript is awesome"

Combining Strings

// Concatenation with + let first = "Hello"; let last = "World"; console.log(first + " " + last); // "Hello World" // concat() method console.log(first.concat(" ", last)); // "Hello World" // Template literals (modern way) console.log(`${first} ${last}`); // "Hello World"

Comparing Strings

// Equality comparison console.log("hello" === "hello"); // true console.log("hello" === "Hello"); // false (case-sensitive) // Lexicographical comparison (Unicode order) console.log("a" < "b"); // true console.log("A" < "a"); // true (uppercase comes first) console.log("10" < "2"); // true (string comparison, not numeric) // localeCompare() - language-aware comparison let str1 = "apple"; let str2 = "banana"; console.log(str1.localeCompare(str2)); // -1 (str1 comes before str2) console.log(str2.localeCompare(str1)); // 1 (str2 comes after str1) console.log(str1.localeCompare(str1)); // 0 (equal)

üîç Regular Expressions

Regular expressions (regex) are patterns used to match character combinations in strings. They're powerful for validation, searching, and text manipulation.

Creating Regular Expressions

// Method 1: Literal notation let regex1 = /hello/; // Method 2: Constructor let regex2 = new RegExp("hello"); // With flags let caseInsensitive = /hello/i; // i = ignore case let global = /hello/g; // g = find all matches let both = /hello/gi; // combine flags

Common Regex Patterns

Pattern Meaning Example
\d Any digit (0-9) /\d{3}/ matches "123"
\w Word character (a-z, A-Z, 0-9, _) /\w+/ matches "hello"
\s Whitespace (space, tab, newline) /\s+/ matches spaces
. Any character except newline /h.t/ matches "hat", "hot"
^ Start of string /^Hello/ matches start
$ End of string /world$/ matches end

Regex Quantifiers

Quantifier Meaning Example
* 0 or more /bo*/ matches "b", "bo", "boo"
+ 1 or more /bo+/ matches "bo", "boo"
? 0 or 1 /colou?r/ matches "color", "colour"
{n} Exactly n times /\d{3}/ matches "123"
{n,} n or more times /\d{3,}/ matches 3+ digits
{n,m} Between n and m /\d{2,4}/ matches 2-4 digits

Character Classes

// [abc] - matches any character in brackets let vowels = /[aeiou]/; console.log(vowels.test("hello")); // true // [^abc] - matches any character NOT in brackets let notVowels = /[^aeiou]/; console.log(notVowels.test("hello")); // true (matches "h", "l", etc.) // [a-z] - range of characters let lowercase = /[a-z]+/; let uppercase = /[A-Z]+/; let digit = /[0-9]+/; // Combine ranges let alphanumeric = /[a-zA-Z0-9]+/;

Regex Methods

let text = "The phone number is 555-123-4567"; // test() - returns true/false let hasNumber = /\d+/.test(text); console.log(hasNumber); // true // search() - returns index of match or -1 let index = text.search(/\d{3}/); console.log(index); // 20 (position of "555") // match() - returns array of matches let matches = text.match(/\d+/g); console.log(matches); // ["555", "123", "4567"] // replace() - replace matches let censored = text.replace(/\d/g, "X"); console.log(censored); // "The phone number is XXX-XXX-XXXX" // exec() - detailed match info let pattern = /\d{3}/g; let result = pattern.exec(text); console.log(result); // ["555", index: 20, input: "...", groups: undefined]

Practical Regex Examples

Email Validation

let email = /^[^\s@]+@[^\s@]+\.[^\s@]+$/; console.log(email.test("user@email.com")); // true console.log(email.test("invalid")); // false

Phone Number

let phone = /^\d{3}-\d{3}-\d{4}$/; console.log(phone.test("555-123-4567")); // true console.log(phone.test("5551234567")); // false

URL Validation

let url = /^https?:\/\/.+/; console.log(url.test("https://google.com")); // true console.log(url.test("ftp://site.com")); // false

Password Strength

// 8+ chars, uppercase, lowercase, digit let hasUpper = /[A-Z]/; let hasLower = /[a-z]/; let hasDigit = /\d/; let minLength = /.{8,}/; function isStrongPassword(pwd) { return hasUpper.test(pwd) && hasLower.test(pwd) && hasDigit.test(pwd) && minLength.test(pwd); }

Extract Numbers

let text = "I have 3 cats and 2 dogs"; let numbers = text.match(/\d+/g); console.log(numbers); // ["3", "2"]

Remove HTML Tags

let html = "<p>Hello</p>"; let clean = html.replace(/<[^>]+>/g, ""); console.log(clean); // "Hello"

üìä Array Methods

Arrays have many built-in methods for manipulation and transformation.

Sorting Arrays

// Default sort (lexicographical) let fruits = ["banana", "apple", "cherry"]; fruits.sort(); console.log(fruits); // ["apple", "banana", "cherry"] // Reverse fruits.reverse(); console.log(fruits); // ["cherry", "banana", "apple"] // Problem: Numbers sort as strings! let numbers = [10, 5, 100, 1]; numbers.sort(); console.log(numbers); // [1, 10, 100, 5] - WRONG! // Solution: Use compare function // Ascending order numbers.sort(function(a, b) { return a - b; }); console.log(numbers); // [1, 5, 10, 100] - CORRECT! // Descending order numbers.sort(function(a, b) { return b - a; }); console.log(numbers); // [100, 10, 5, 1] // Modern arrow function syntax numbers.sort((a, b) => a - b); // Ascending numbers.sort((a, b) => b - a); // Descending
üí° Compare Function Logic:
  • Return negative: a comes before b
  • Return positive: b comes before a
  • Return 0: order unchanged

Extracting and Modifying

let arr = [1, 2, 3, 4, 5]; // slice(start, end) - extract without modifying let subarray = arr.slice(1, 4); console.log(subarray); // [2, 3, 4] console.log(arr); // [1, 2, 3, 4, 5] - unchanged // splice(start, deleteCount, items...) - modify array let arr2 = [1, 2, 3, 4, 5]; // Remove 2 elements starting at index 1 let removed = arr2.splice(1, 2); console.log(removed); // [2, 3] console.log(arr2); // [1, 4, 5] // Insert elements arr2.splice(1, 0, 10, 20); // Insert at index 1 console.log(arr2); // [1, 10, 20, 4, 5] // Replace elements arr2.splice(1, 2, 99); // Remove 2, insert 99 console.log(arr2); // [1, 99, 4, 5]

Stacks and Queues

Stack (LIFO) - Last In, First Out:

let stack = []; // Add to top (end) stack.push(1); stack.push(2); stack.push(3); console.log(stack); // [1, 2, 3] // Remove from top (end) let last = stack.pop(); console.log(last); // 3 console.log(stack); // [1, 2]

Queue (FIFO) - First In, First Out:

let queue = []; // Add to end queue.push(1); queue.push(2); queue.push(3); console.log(queue); // [1, 2, 3] // Remove from beginning let first = queue.shift(); console.log(first); // 1 console.log(queue); // [2, 3] // Add to beginning queue.unshift(0); console.log(queue); // [0, 2, 3]

Array Method Summary

Method Description Modifies?
push() Add to end Yes
pop() Remove from end Yes
unshift() Add to beginning Yes
shift() Remove from beginning Yes
slice() Extract subarray No
splice() Remove/insert elements Yes
sort() Sort array Yes
reverse() Reverse order Yes

üî¢ Math Object

The Math object provides mathematical constants and functions.

Common Math Methods

// Rounding console.log(Math.round(4.7)); // 5 console.log(Math.round(4.4)); // 4 console.log(Math.ceil(4.1)); // 5 (round up) console.log(Math.floor(4.9)); // 4 (round down) console.log(Math.trunc(4.9)); // 4 (remove decimals) // Absolute value console.log(Math.abs(-5)); // 5 // Power and roots console.log(Math.pow(2, 3)); // 8 (2¬≥) console.log(Math.sqrt(16)); // 4 console.log(Math.cbrt(27)); // 3 (cube root) // Min and max console.log(Math.min(1, 5, 3)); // 1 console.log(Math.max(1, 5, 3)); // 5 // Min/max of array let numbers = [1, 5, 3, 9, 2]; console.log(Math.max(...numbers)); // 9 (spread operator) console.log(Math.min(...numbers)); // 1

Math Constants

console.log(Math.PI); // 3.141592653589793 console.log(Math.E); // 2.718281828459045 console.log(Math.LN2); // 0.6931471805599453 console.log(Math.LN10); // 2.302585092994046 console.log(Math.SQRT2); // 1.4142135623730951

Random Numbers

// Random number between 0 and 1 let random = Math.random(); console.log(random); // e.g., 0.7234... // Random integer from 0 to 9 let dice = Math.floor(Math.random() * 10); // Random integer from 1 to 6 (dice) let roll = Math.floor(Math.random() * 6) + 1; // Random integer between min and max (inclusive) function randomInt(min, max) { return Math.floor(Math.random() * (max - min + 1)) + min; } console.log(randomInt(1, 100)); // Random number 1-100 // Shuffle array randomly let arr = [1, 2, 3, 4, 5]; arr.sort(() => Math.random() - 0.5); console.log(arr); // Randomly shuffled

üìÖ Date Object

The Date object works with dates and times.

Creating Dates

// Current date and time let now = new Date(); console.log(now); // From string let date1 = new Date("2024-12-25"); let date2 = new Date("December 25, 2024"); // From numbers (year, month, day, hour, min, sec, ms) // Note: Month is 0-indexed (0 = January, 11 = December) let date3 = new Date(2024, 11, 25); // Dec 25, 2024 let date4 = new Date(2024, 11, 25, 14, 30, 0); // 2:30 PM // From timestamp (milliseconds since Jan 1, 1970) let date5 = new Date(1609459200000);

Getting Date Components

let date = new Date(2024, 11, 25, 14, 30, 45); console.log(date.getFullYear()); // 2024 console.log(date.getMonth()); // 11 (December) console.log(date.getDate()); // 25 (day of month) console.log(date.getDay()); // 3 (Wed: 0=Sun, 6=Sat) console.log(date.getHours()); // 14 console.log(date.getMinutes()); // 30 console.log(date.getSeconds()); // 45 console.log(date.getTime()); // Timestamp in ms

Setting Date Components

let date = new Date(); date.setFullYear(2025); date.setMonth(0); // January date.setDate(1); // 1st day date.setHours(12); date.setMinutes(0); date.setSeconds(0); console.log(date); // Jan 1, 2025, 12:00:00

Date Formatting

let date = new Date(); // Default string console.log(date.toString()); // "Wed Dec 25 2024 14:30:45 GMT-0500" // Date only console.log(date.toDateString()); // "Wed Dec 25 2024" // Time only console.log(date.toTimeString()); // "14:30:45 GMT-0500" // Locale format console.log(date.toLocaleString()); // "12/25/2024, 2:30:45 PM" console.log(date.toLocaleDateString()); // "12/25/2024" console.log(date.toLocaleTimeString()); // "2:30:45 PM" // ISO format console.log(date.toISOString()); // "2024-12-25T19:30:45.000Z"

Date Calculations

// Days between dates let date1 = new Date("2024-01-01"); let date2 = new Date("2024-12-31"); let diff = date2 - date1; // Difference in milliseconds let days = diff / (1000 * 60 * 60 * 24); console.log(days); // 365 // Add days to date let today = new Date(); let tomorrow = new Date(today); tomorrow.setDate(tomorrow.getDate() + 1); // Countdown timer function daysUntil(targetDate) { let now = new Date(); let target = new Date(targetDate); let diff = target - now; return Math.ceil(diff / (1000 * 60 * 60 * 24)); } console.log(daysUntil("2025-01-01"));

üìú Template Literals

Template literals provide a more powerful way to work with strings.

Basic Template Literals

// Use backticks instead of quotes let greeting = `Hello World`; // Multi-line strings let multiline = `This is a multi-line string`; // No need for escaping quotes let quote = `He said "Hello" and I said 'Hi'`;

Placeholders (String Interpolation)

// Embed variables let name = "John"; let age = 30; console.log(`My name is ${name} and I am ${age} years old`); // Embed expressions let price = 19.99; let quantity = 3; console.log(`Total: $${price * quantity}`); // "Total: $59.97" // Call functions function formatName(name) { return name.toUpperCase(); } console.log(`Hello, ${formatName(name)}!`); // "Hello, JOHN!" // Conditional expressions let score = 85; console.log(`You ${score >= 60 ? "passed" : "failed"}`);

Practical Examples

// HTML generation let user = {name: "Alice", email: "alice@email.com"}; let html = ` <div class="user"> <h2>${user.name}</h2> <p>${user.email}</p> </div> `; // URL building let userId = 123; let url = `https://api.example.com/users/${userId}`; // SQL queries (use with caution!) let username = "john"; let query = `SELECT * FROM users WHERE name = '${username}'`; // Messages let items = 5; console.log(`You have ${items} item${items !== 1 ? "s" : ""} in your cart`);

üí° Best Practices

  1. Use const for regex patterns
    const EMAIL_PATTERN = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  2. Test regex patterns thoroughly
  3. Use template literals for complex strings
  4. Remember: strings are immutable
  5. Use arrow functions for compare functions
    arr.sort((a, b) => a - b);
  6. Cache Date objects when doing multiple calculations
  7. Use spread operator for array operations
