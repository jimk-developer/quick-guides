A Beginner's Guide to Arrays

üéØ What are Arrays?

An array is a special type of variable that can hold multiple values in a single container. Think of an array as a numbered list where each item has a position (called an index).

Array Structure Visualization

let fruits = ["apple", "banana", "orange"];

"apple"

"banana"

"orange"
Index: 2
üí° Key Array Concepts:
  • Zero-indexed - Arrays start counting at 0, not 1
  • Ordered - Items stay in the order you put them
  • Dynamic size - Arrays can grow or shrink as needed
  • Mixed types - Can hold different data types (though usually you use one type)

üìù Creating Arrays

There are multiple ways to create arrays in JavaScript.

Array Literal (Most Common)

// Empty array let emptyArray = []; // Array with numbers let numbers = [1, 2, 3, 4, 5]; // Array with strings let colors = ["red", "green", "blue"]; // Array with mixed types (possible but not recommended) let mixed = [42, "hello", true]; // Array with multiple elements let weekdays = [ "Monday", "Tuesday", "Wednesday", "Thursday", "Friday" ];

Array Constructor

// Using the Array constructor let arr1 = new Array(3); // Creates array with 3 empty slots let arr2 = new Array(1, 2, 3); // Creates [1, 2, 3] // Array literal is preferred for simplicity let preferred = [1, 2, 3];
üí° Best Practice: Use array literals [] instead of the Array constructor. It's simpler and less confusing.

üîç Accessing Array Elements

Access array elements using square brackets and the index number. Remember: arrays are zero-indexed!

let fruits = ["apple", "banana", "orange", "grape"]; // Accessing individual elements console.log(fruits[0]); // Output: "apple" console.log(fruits[1]); // Output: "banana" console.log(fruits[2]); // Output: "orange" console.log(fruits[3]); // Output: "grape" // Storing an element in a variable let firstFruit = fruits[0]; console.log(firstFruit); // Output: "apple" // Accessing out of bounds returns undefined console.log(fruits[10]); // Output: undefined

Array Length

The length property tells you how many elements are in an array.

let colors = ["red", "green", "blue"]; console.log(colors.length); // Output: 3 // Access the last element let lastColor = colors[colors.length - 1]; console.log(lastColor); // Output: "blue" // Empty array has length 0 let empty = []; console.log(empty.length); // Output: 0
‚ö†Ô∏è Important: Array length is always 1 more than the highest index because arrays start at 0. An array with 5 elements has indices 0-4.

‚úèÔ∏è Modifying Arrays

Changing Elements

You can change any element by assigning a new value to that index.

let pets = ["dog", "cat", "bird"]; // Change the second element pets[1] = "hamster"; console.log(pets); // Output: ["dog", "hamster", "bird"] // Change the first element pets[0] = "fish"; console.log(pets); // Output: ["fish", "hamster", "bird"]

‚ûï Adding Elements

push() - Add to End

The push() method adds one or more elements to the end of an array.

let numbers = [1, 2, 3]; // Add one element numbers.push(4); console.log(numbers); // Output: [1, 2, 3, 4] // Add multiple elements numbers.push(5, 6, 7); console.log(numbers); // Output: [1, 2, 3, 4, 5, 6, 7] // push() returns the new length let newLength = numbers.push(8); console.log(newLength); // Output: 8

unshift() - Add to Beginning

The unshift() method adds one or more elements to the beginning of an array.

let letters = ["b", "c", "d"]; // Add to the beginning letters.unshift("a"); console.log(letters); // Output: ["a", "b", "c", "d"] // Add multiple elements letters.unshift("x", "y", "z"); console.log(letters); // Output: ["x", "y", "z", "a", "b", "c", "d"]

‚ûñ Removing Elements

pop() - Remove from End

The pop() method removes and returns the last element from an array.

let fruits = ["apple", "banana", "orange"]; // Remove the last element let removed = fruits.pop(); console.log(removed); // Output: "orange" console.log(fruits); // Output: ["apple", "banana"] // Remove another fruits.pop(); console.log(fruits); // Output: ["apple"]

shift() - Remove from Beginning

The shift() method removes and returns the first element from an array.

let colors = ["red", "green", "blue"]; // Remove the first element let removed = colors.shift(); console.log(removed); // Output: "red" console.log(colors); // Output: ["green", "blue"] // Remove another colors.shift(); console.log(colors); // Output: ["blue"]

Summary of Add/Remove Methods

Method What It Does Returns Example
push() Adds to end New length arr.push(5)
pop() Removes from end Removed element arr.pop()
unshift() Adds to beginning New length arr.unshift(1)
shift() Removes from beginning Removed element arr.shift()

üîÑ Common Array Methods

indexOf() - Find Position

Returns the index of the first occurrence of an element, or -1 if not found.

let animals = ["cat", "dog", "bird", "cat"]; console.log(animals.indexOf("dog")); // Output: 1 console.log(animals.indexOf("cat")); // Output: 0 (first occurrence) console.log(animals.indexOf("fish")); // Output: -1 (not found)

includes() - Check if Element Exists

Returns true if the array contains the element, false otherwise.

let fruits = ["apple", "banana", "orange"]; console.log(fruits.includes("banana")); // Output: true console.log(fruits.includes("grape")); // Output: false

slice() - Copy Portion

Returns a shallow copy of a portion of an array. Does not modify the original array.

let numbers = [1, 2, 3, 4, 5]; // Get elements from index 1 to 3 (not including 3) let sliced = numbers.slice(1, 3); console.log(sliced); // Output: [2, 3] console.log(numbers); // Output: [1, 2, 3, 4, 5] (unchanged) // Get from index 2 to end let sliced2 = numbers.slice(2); console.log(sliced2); // Output: [3, 4, 5] // Copy entire array let copy = numbers.slice(); console.log(copy); // Output: [1, 2, 3, 4, 5]

concat() - Combine Arrays

Combines two or more arrays into a new array. Does not modify the original arrays.

let arr1 = [1, 2, 3]; let arr2 = [4, 5, 6]; let arr3 = [7, 8, 9]; // Combine two arrays let combined = arr1.concat(arr2); console.log(combined); // Output: [1, 2, 3, 4, 5, 6] // Combine multiple arrays let all = arr1.concat(arr2, arr3); console.log(all); // Output: [1, 2, 3, 4, 5, 6, 7, 8, 9] // Original arrays unchanged console.log(arr1); // Output: [1, 2, 3]

join() - Convert to String

Joins all array elements into a single string, separated by a specified separator.

let words = ["Hello", "World", "JavaScript"]; // Join with space let sentence = words.join(" "); console.log(sentence); // Output: "Hello World JavaScript" // Join with comma let list = words.join(", "); console.log(list); // Output: "Hello, World, JavaScript" // Join with no separator let combined = words.join(""); console.log(combined); // Output: "HelloWorldJavaScript" // Default separator is comma console.log(words.join()); // Output: "Hello,World,JavaScript"

reverse() - Reverse Order

Reverses the order of elements in an array. This modifies the original array.

let numbers = [1, 2, 3, 4, 5]; numbers.reverse(); console.log(numbers); // Output: [5, 4, 3, 2, 1] let letters = ["a", "b", "c"]; letters.reverse(); console.log(letters); // Output: ["c", "b", "a"]

sort() - Sort Elements

Sorts the elements of an array. This modifies the original array. By default, sorts as strings.

// Sorting strings (alphabetically) let fruits = ["banana", "apple", "orange", "grape"]; fruits.sort(); console.log(fruits); // Output: ["apple", "banana", "grape", "orange"] // Sorting numbers requires a compare function let numbers = [10, 5, 20, 3, 15]; numbers.sort(function(a, b) { return a - b; // Ascending order }); console.log(numbers); // Output: [3, 5, 10, 15, 20]
‚ö†Ô∏è Warning: sort() without a compare function converts numbers to strings. [10, 5, 20] becomes [10, 20, 5] because "10" comes before "5" alphabetically!

üìö Practical Examples

Example 1: Shopping List

let shopping = ["milk", "eggs", "bread"]; // Add items shopping.push("butter"); shopping.push("cheese"); console.log(shopping); // ["milk", "eggs", "bread", "butter", "cheese"] console.log("Items: " + shopping.length); // Items: 5

Example 2: Student Grades

let grades = [85, 92, 78, 90, 88]; // Add a new grade grades.push(95); // Get first and last grades console.log("First: " + grades[0]); console.log("Last: " + grades[grades.length - 1]);

Example 3: Playlist

let playlist = ["Song A", "Song B", "Song C"]; // Add to end playlist.push("Song D"); // Add to beginning playlist.unshift("Intro"); console.log(playlist); // ["Intro", "Song A", "Song B", "Song C", "Song D"]

Example 4: Favorite Colors

let colors = ["blue", "red", "green"]; // Check if color exists console.log(colors.includes("red")); // true console.log(colors.includes("yellow")); // false // Find position console.log(colors.indexOf("green")); // 2

Example 5: Todo List

let todos = ["Study", "Exercise", "Code"]; // Complete first task (remove it) let completed = todos.shift(); console.log("Completed: " + completed); // Add new task todos.push("Read"); console.log(todos); // ["Exercise", "Code", "Read"]

Example 6: Combining Lists

let morningClasses = ["Math", "English"]; let afternoonClasses = ["Science", "History"]; let allClasses = morningClasses.concat(afternoonClasses); console.log(allClasses); // ["Math", "English", "Science", "History"]

üí° Best Practices

  1. Use descriptive names - Array names should be plural nouns
    // ‚ùå Bad let x = [1, 2, 3]; // ‚úÖ Good let scores = [1, 2, 3];
  2. Keep arrays homogeneous - Store the same type of data in each array
  3. Use const for arrays you don't reassign - You can still modify the contents
    const fruits = ["apple", "banana"]; fruits.push("orange"); // ‚úÖ This works // fruits = []; // ‚ùå This would cause an error
  4. Check array length before accessing - Avoid accessing indices that don't exist
  5. Use array methods instead of manual loops - Methods like push, pop are more readable
  6. Remember arrays are zero-indexed - First element is at index 0

üö´ Common Mistakes

1. Forgetting Arrays Start at 0

let numbers = [10, 20, 30]; // ‚ùå Wrong - trying to get "first" with 1 console.log(numbers[1]); // Output: 20 (second element!) // ‚úÖ Correct - first element is at 0 console.log(numbers[0]); // Output: 10

2. Modifying Array While Getting Length

let items = ["a", "b", "c"]; // Length is 3, but last index is 2 // console.log(items[items.length]); // ‚ùå undefined // ‚úÖ Correct - subtract 1 for last index console.log(items[items.length - 1]); // "c"

3. Confusing push/pop with shift/unshift

let nums = [1, 2, 3]; // push/pop work on the END nums.push(4); // [1, 2, 3, 4] nums.pop(); // [1, 2, 3] // shift/unshift work on the BEGINNING nums.unshift(0); // [0, 1, 2, 3] nums.shift(); // [1, 2, 3]

4. Forgetting That Some Methods Modify the Array

let original = [3, 1, 2]; // reverse() and sort() MODIFY the original original.reverse(); console.log(original); // [2, 1, 3] - CHANGED! // slice() and concat() DO NOT modify the original let nums = [1, 2, 3]; let sliced = nums.slice(0, 2); console.log(nums); // [1, 2, 3] - unchanged

üìù Key Takeaways:
  • Arrays store multiple values in a single variable
  • Arrays are zero-indexed (first element is at index 0)
  • Use length property to get the number of elements
  • Use push() and pop() for end operations
  • Use unshift() and shift() for beginning operations
  • Some methods modify the array, others return new arrays
  • Arrays can grow and shrink dynamically
  • Access elements with square brackets: array[index]
